HF Vertical Designer – Quick Start Guide

This Quick Start Guide walks you through the fastest way to use the HF Vertical Designer. In just a few steps, you’ll be modeling vertical antennas, checking SWR across all HF bands, and estimating ERP for any configuration.

1. Enter Your Antenna Length

In the Antenna Geometry panel:

  • Set Element length (ft) to the wire you plan to deploy.
  • Set Mast height to your support pole height.
  • Choose Foldover or Helical if the wire exceeds mast height.
  • Enable Counterpoise if using EFHW or random-wire systems.

Examples:

  • 25 ft → Classic Rybakov
  • 29 ft → Random-wire 9:1 vertical
  • 33 ft → 20m EFHW
  • 41 ft → 80–10m long vertical

2. Choose Transformer / UNUN

Select the matching transformer:

  • 4:1 → 17.5 ft, 20.5 ft, 25 ft Rybakov family
  • 9:1 → 29 ft, 41 ft, helical 33 ft
  • 49:1 → EFHW (20m, 40m)
  • 64:1 → EFHW (15m optimal)
  • 1:1 → Vertical dipoles / OCF vertical dipoles

3. Configure Feedline

Choose your feedline type and length:

  • LMR‑400 → lowest loss
  • RG‑213 → field standard
  • RG‑8X → portable-friendly
  • RG‑58 → ultralight, highest loss
  • 450Ω ladder line → doublets / OCF systems

The calculator automatically computes coax loss and electrical length.

4. Set Environment & Radials

Choose your operating environment:

  • Saltwater → +3 dB low-angle gain
  • Good ground → +1 dB
  • Inland soil → baseline

Enter radial count and length if using a ground-mounted vertical.

5. Press “Calculate Bands & ERP”

The results populate instantly:

  • Band Capability Table — usable, tuner-only, or not matchable
  • SWR (no tuner) — raw feedpoint + transformer + coax
  • SWR (with tuner) — estimated ATU-corrected SWR
  • ERP — effective radiated power per band
  • Geometry Summary — effective height, foldover, counterpoise
  • Feedline Summary — loss, VF, electrical length
  • Radiation Plots — azimuth, elevation, donut

6. Tips for Best Results

  • Use 25 ft or 29 ft for broad multi-band performance.
  • Use 33 ft EFHW for 20m DX dominance.
  • Use 41 ft for 80m–10m coverage.
  • Deploy near saltwater whenever possible.
  • Use 4–7 radials for Rybakov/random-wire systems.
